# 🏗️ 건설 수주 · 재개발 뉴스 트래커

건설사별 **신규 수주**, **재개발·재건축**, **분양** 뉴스를  
네이버 뉴스 RSS와 Google 뉴스 RSS에서 실시간 수집하는 Streamlit 앱입니다.

## 실행 방법

```bash
# 1. 패키지 설치
pip install -r requirements.txt

# 2. 앱 실행
streamlit run app.py
```

## 주요 기능

| 기능 | 설명 |
|------|------|
| 건설사 선택 | 10개 대형 건설사 + 직접 추가 가능 |
| 추가 키워드 | 자유롭게 키워드 입력 |
| 카테고리 분류 | 재개발·재건축 / 신규수주 / 분양 자동 분류 |
| 탭 뷰 | 카테고리별 탭으로 빠른 탐색 |
| CSV 내보내기 | 수집 결과 엑셀/CSV 다운로드 |

## 수집 출처

- **네이버 뉴스 RSS** — `https://search.naver.com/rss.nhn?where=news&query=...`
- **Google 뉴스 RSS** — `https://news.google.com/rss/search?q=...&hl=ko&gl=KR`
